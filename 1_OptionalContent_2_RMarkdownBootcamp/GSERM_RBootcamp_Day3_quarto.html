Base Graphics

The base graphics system in R works like an artist’s canvas. You start with a blank canvas and build up your graph using annotation functions to add or modify elements such as text and lines. In this code chunk, we explore various base graphics functions. We create a histogram of tips using hist(), and then use the plot() function to create scatterplots of tips against different variables.

hist(tip) #let's examine the distribution of tips

#the plot function is "polymorphic", i.e. it produces different
#results given the data type - see the following examples
plot(size,tip) #plotting number of people (data type = numeric) vs tips

plot(sex,tip) #tips by sex (data type = factor)

plot(day,tip) #tips by day (data type = factor)

Now, let’s customize labels, titles, and margins using par() and add reference lines using abline().

#customize labels, titles etc.
par(mar=c(5,5,5,5))#adjust plot margins
plot(size,tip,main="Party Size on Tips",xlab="Party Size",ylab="Tips (in $)",cex.lab=1.8)
abline(v=mean(size)) #add a vertical line based on mean(size)
abline(h=mean(tip)) #add a horizontal line based on mean(tip)

Looks like a linear trend, right? Let’s fit a linear model using lm() and add the regression line to the plot.

mod <- lm(tip~size) #first, create an object of the lm() function

Lattice Graphics

Lattice graphics are particularly useful for creating conditioning plots, where you can see how the relationship between variables changes over different levels of a conditioning variable. In this code chunk, we explore lattice graphics using functions like histogram(), xyplot(), and bwplot(). We create conditioning plots using the | operator to split the data by different levels of a conditioning variable. We also demonstrate how to create a custom panel function to add smoothing splines and linear model fits to the plots.

histogram(tip)

xyplot(tip~size) #simple bivariate plot

#The power of the lattice package is within conditioning plots
#Here are some examples:
xyplot(tip~size|sex)

xyplot(tip~size|time*day)

bwplot(tip~day|sex)

bwplot(tip~sex|day)

#Now a more complex example where we setup our own function 
#PANEL function - here with a smoothing spline and linear model fit 
#day <- factor(day,levels=c("Thur","Fri","Sat","Sun")) #optional: reordering of a factor
#time <- factor(time,levels=c("Lunch","Dinner")) #optional: reordering of a factor
xyplot(tip~size|sex,
    layout=c(2,1), #adjust layout if you want the panels to be horizontally aligned
    panel=function(x,y,...){
    panel.xyplot(x,y,...)
    panel.loess(x,y,col="black",lwd=2,lty=2)
    mod <- lm(y~x)
    panel.abline(mod,lwd=2,col="red")
})

ggplot2 Graphics

ggplot2 is a powerful graphics package that follows the “grammar of graphics” philosophy. It maps data to aesthetic attributes (e.g., color) of geometric objects (e.g., points, lines) and may contain statistical transformations. In this code chunk, we explore ggplot2 graphics using the qplot() function. We create histograms, density plots, and scatterplots with overlays based on factor variables. We also demonstrate how to create multiple plots in parallel using the gridExtra package and how to use the GGally package for creating matrix plots of pairwise relationships between variables during the exploratory data analysis stage of a project.

Statistical Testing & Model Building

In this section, we will perform statistical tests and build linear models using the tips dataset. We perform various statistical tests and model building techniques. We start with a t-test to compare tipping between males and females using t.test(). We then perform an ANOVA to compare tips across different days using aov() and explore pairwise comparisons using TukeyHSD(). We also check for normality assumptions using shapiro.test() and transform the data using log() to address violations.
